# PDF Tool

A versatile utility for manipulating and extracting data from PDF documents.

## When to Use This Skill

- Extracting text or data tables from reports
- Merging multiple PDF documents into one
- Splitting a large PDF into separate pages or chapters
- Adding watermarks or annotations

## How to Use

**User**: "Extract the table on page 5 of this report"

**User**: "Merge these three contracts into a single PDF"

**User**: "Convert this PDF to text"
