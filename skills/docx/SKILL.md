# Docx Tool

A tool for programmatically interacting with Microsoft Word documents.

## When to Use This Skill

- Generating contracts or letters from templates
- Extracting comments and revisions from a reviewed document
- Converting documentation to Word format
- analyzing document structure and content

## How to Use

**User**: "Create a new proposal document based on this text content"

**User**: "List all the comments in this agreement file"
