# PPTX Tool

This skill allows for the programmatic creation and modification of PowerPoint presentations.

## When to Use This Skill

- Generating slide decks from markdown or outlines
- Updating templates across multiple presentations
- Extracting text or notes from slides
- Automating report generation into slide format

## How to Use

**User**: "Create a 5-slide presentation about our Q3 goals"

**User**: "Extract all speaker notes from this deck"
